# Gopherus
***
### [Gopherus](https://github.com/tarunkant/Gopherus) is a tool used to generate gopher URLs
#### Services supported:
* MySQL
* PostgreSQL
* FastCGI
* Redis
* SMTP
* Zabbix
* pymemcache
* rbmemcache
* phpmemcache
* dmpmemcache

#### Installling Gopherus
```bash
$ git clone https://github.com/tarunkant/Gopherus
$ cd Gopherus
$ ./install.sh
```

#### Running Gopherus
```bash
$ gopherus -h
```
#### Generate valid SMTP URL
```bash
$ gopherus --exploit smtp
```
